# klimarisk
Master thesis



**Creating Vite React project:**
```powershell

npm create vite

> npx
> cva

│
◇  Project name:
│  frontend
│
◇  Select a framework:
│  React
│
◇  Select a variant:
│  TypeScript + SWC
│
◇  Scaffolding project in C:\Users\[...]\klimarisk\frontend...
│
└  Done. Now run:

  cd frontend
  npm install
  npm run dev

```
